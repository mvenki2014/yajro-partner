# Yajro Priests - Partner App

Cross-platform partner mobile app for Hindu priests (Purohits / Pandits) built with React + Vite + Tailwind + Capacitor.

## What This App Covers

- Priest login and registration (multi-step)
- OTP-based auth flow
- Manage services and availability
- Handle bookings/orders
- Earnings and payout-related UI
- Profile management

## Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Capacitor 8 (Android + iOS)
- React Hook Form + Zod validation
- React Hot Toast

## Scripts

- `npm run dev` - start local dev server
- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run cap:sync` - build and sync Capacitor
- `npm run cap:android` - open Android project
- `npm run cap:ios` - open iOS project

## App Identity

- App Name: `Yajro Partner`
